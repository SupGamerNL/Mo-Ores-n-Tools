# Mo’ Ores ’n Tools

A Minecraft mod that expands vanilla with meaningful ores, tools, armor, and simple processing.

## About

This mod is developed using MCreator for NeoForge (Minecraft 1.21.1). It focuses on:

- Interesting ore diversity
- Tool and armor progression
- Simple crafting and processing
- Exploration incentives

## Roadmap

### Version 1.0
- Tiered ores (Tin → Bronze → Steel → …)
- New tools and armor per ore
- Simple processing items (Hammer, Press, etc.)
- Biome-linked generation

### Version 1.1
- Decorative blocks
- More ore variants
- Refinement mechanics
